# Adidas Sales Dashboard (Power BI)

## Overview
This project presents an interactive Adidas Sales Dashboard built using Power BI.
It analyzes sales performance across regions, products, sales channels, and retailers
to uncover trends, seasonality, and revenue drivers.

## Key KPIs
- Total Sales
- Units Sold
- Price per Unit
- Operating Margin
- Operating Profit

## Key Insights
- Sales show strong seasonality, peaking in July–September and December.
- West region is the highest revenue contributor.
- Online and In-store channels dominate the sales mix.
- Men’s Street Footwear is the top-performing product category.
- Revenue is concentrated among top retailers like Foot Locker and Sports Direct.

## Dashboard Features
- Interactive slicers for Region, Year, and Product
- Monthly sales trend analysis
- Regional sales distribution
- Sales channel contribution
- Product and retailer performance comparison
- Business-focused insight panel

## Tools & Skills Used
- Power BI
- Data Modeling
- DAX
- Data Visualization
- Retail & Sales Analytics

## How to Use
1. Download the `.pbix` file from the dashboard folder.
2. Open it in Power BI Desktop.
3. Use slicers to explore sales insights interactively.
